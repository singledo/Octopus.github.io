deb 解压,修改,压缩

deb 文件的解压

deb 文件的解压

> sudo dpkg -e file.deb extract/DEBIAN

修改

修改依赖文件
> vim extract/DEBIAN/control

压缩

deb 文件的压缩
> sudo dpkg -b extract/build

安装

deb 文件的安装
> sudo dpkg -i file.deb
