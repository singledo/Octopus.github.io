算法导论-第六章(堆排序)

堆的基本概念

  • 堆数据结构是一种数组对象 ，可以被视为一颗完全二叉树 。
  • 树除了最后一层 ，其他层都是填满的 。
  • length[A] 是数组中的元素的个数 ，heap-size[A] 是存放在 A 中的堆的元素个数 。heap-size[A] ≤length[A] 。
  • 给定了某个节点 i ，left(i) = 2i ， right(i) = 2i + 1 ，parent(i) = CEIL(i) ；
  • 最大堆： A[parent(i)] ≥ A[i] ，最小堆：A[parent(i)] ≤A[i] 。
  • 堆的高度为树根的高度 。

堆排序的优先级 - 优先级队列

优先级队列是用来未付一组由元素构成的集合 S 的数据结构 ，这一组元素中的每一个都有一个关键字 key 。一个最大优先级队列支持一下操作
1. INSERT(S,x); 将元素 x 插入集合 S 。
2. MAXINUM(S); 返回 S 中的最大关键字的元素 。 
3. EXTRACT-MAX(S); 去掉并返回 S 中的具有最大关键字的元素 。
4. INCREASE-KEY(S,x,k); 将元素 x 的关键字的值增加到 k , 这里的 k 值不能小于 x 的原关键字值。
