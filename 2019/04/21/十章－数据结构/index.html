第十章－基本的数据结构

第十章 — 基本的数据结构

动态集合的元素

动态集合的对象为一个关键字的的集合。所有的关键字都取自与一个全序集　。

动态集合上的操作

动态集合上的操作一般为两类操作：1. 查询操作　。2. 修改操作 。下面的为一些常见的操作

1. 给定一个集合 S 和关键字 k ，返回一个指向 S 中一个对象的指针 p ;　或者当　k 不存在时 , 返回 nil .
    SEARCH(S,k)

2. 将 x　指向的对象插入到集合 S 中去 . 
    INSERT(S,x)

3. 当给定一个指向 S 中的元素的指针 x 时 , 将 x 从 S 中删除 . 
    DELETE(S,x)

4. 获取指定集合 S 中的最小值 .
    MININUM(S)

5. 获取指定集合 S 中的最大值 .
    MAXNUM(S)

6.　获取指定集合 S 中比 x 大的下一个元素 . 
    SUCCESSOR(S, x)

7. 获取指定集合 S 中比 x 小的上一个元素 . 
    PREDECESSOR(S, x)

栈及队列

栈: 一种后进先出的数据结构 , 称为 LIFO ( last in, first out ) .　栈的删除只能删除最后插入的对象 ,  称之为入栈 ( POP ) . 栈的插入只能在最后插入的对象后面插入 , 称之为入栈 ( PUSH ) .
栈下溢: 对空栈进行删除操作.
栈上溢: 对满栈进行插入操作.

队列: 一种先进先出的数据结构 , 称为 FIFO (first in, first out ) . 队列的删除在队列头部，称为出队 (DEQUEUE) ,　队列的插入在队列尾部 , 称为入队 (EnQUEUE) .
队列下溢: 对空队列进行删除操作.
队列上溢: 对满队列进行插入操作.

两个栈实现一个队列 ?

我们将两个栈看成最后实现的队列,此时将实现看成黑盒子. 我们向队列中插入 1 到 7 ,　依次取数据为 1　到 7 .

此时我们有两个栈S1, s2, 

   S1 +-----+  top     S2 +----+   top 
      |     |             |    |
      |     |             |    |
      |     |             |    |
      |     |             |    |
      |     |             |    |
      |     |             |    |
      |     |             |    |
 bottom               bottom

 step 1: 将需要插入队列的对象插入 S1 , 我们假设为 1~7 , 下面标示为插入后的两个栈的数据分布情况.

   S1 +-----+  top     S2 +----+   top 
      |  1  |             |    |
      |  2  |             |    |
      |  3  |             |    |
      |  4  |             |    |
      |  5  |             |    |
      |  6  |             |    |
      |  7  |             |    |
 bottom               bottom

 step 2: 出栈时，先将 S1 中的数据安找入栈的顺序依次入栈 S2 . 并将 S2 栈底数据出栈.

   S1 +-----+  top     S2 +-----+   top 
      |     |             |  7  |
      |     |             |  6  |
      |     |             |  5  |
      |     |             |  4  |
      |     |             |  3  |
      |     |             |  2  |
      |     |             |  1  | ---- this node  will pop
 bottom               bottom

 Step 3: 完成数据弹出后, 按照 S2 的出栈的顺序入栈 S1 .

   S1 +-----+  top     S2 +-----+   top 
      |  2  |             |     |
      |  3  |             |     |
      |  4  |             |     |
      |  5  |             |     |
      |  6  |             |     |
      |  7  |             |     |
 bottom               bottom


step 2 中, S1 的最后一个元素不必插入到 S2 中 , 直接弹出即可 . 
step 3 中，完成队列的出栈后,　不必立即将 S2 数据弹出到 S1 . 如果对于本队列的下一次操作为弹出，直接在 S2 中操作即可 . 如果是插入操作，则将 S2 中数据按照 出栈顺序插入 S1 .

两个队列实现栈 ?

我们假设插入的数据为 1~7, 我们取出数据的顺序为 7 -> 1.

假设我们有两个队列 q1, q2.

    q1                        q2
    end        head         end        head
    +-------------            +-------------
    |                       |              
    +-------------            +-------------

step 1: 将数据按照顺序插入到 q1 中 , 下面为数据在两个队列中的分布情况.

    q1                        q2
    end        head         end        head
    +-------------            +-------------
    |7 6 5 4 3 2 1          |              
    +-------------            +-------------

step 2: 除了在队末的数据，将在 q1 中的数据出列到 q2 中, 将 q1 中的数据出列,　完成数据的出栈.

    q1                        q2
    end        head         end        head
    +-------------            +-------------
    |7(DEQEUE)              |6 5 4 3 2 1    
    +-------------            +-------------

step 3: 将 q2 中的数据全部出列，并按照 q2 出列顺序入列 q1.

    q1                        q2
    end        head         end        head
    +-------------            +-------------
    |6 5 4 3 2 1           |               
    +-------------            +-------------

链表

双向链表: 每一个元素都包含一个对象, 每个对象中包含一个关键字域(key)和两个指针域(next, prev), next[x] 指向链表中的 x 元素的后继元素, prev[x] 标示链表中 x 的前驱元素. 当 x　元素没有前驱元素时，此元素为head, 当元素没有后继元素时, 此元素为 tail.

链表的基本操作

  • 链表的搜索

    LIST-SEARCH(L, k)

    x ← head[L]
while x != nil and key[x] != k
    do x ← next[x]
return x

  • 链表的插入
    LIST-INSERT(L,new)

    x ← head[L]
while x != nil
    x ← next[x]
next[x] ← new
prev[nex] ← x

  • 链表的删除
    List-DELETE(L,k)

    x ← head[L]
while x != nil and key[x] != k
    x ← next[x]
if x == nil
    return nil
delNode ← x
prev[delNode] ← next[delNode]
prev[next[delNode]] ← prev[delNode]
