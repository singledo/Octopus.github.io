十五章-动态规划

第十五章 - 动态规划

  1. 动态规划的适用情况
    问题可以分割成若干子问题 ，而且子问题并非是独立的 ，各个子问题包含相同的子子问题 ，对于相同的子子问题只会求解一次 ，将其保存在一张表中 。
  2. 动态规划算法设计
    a. 描述最优解的结构 。
    b. 递归定义最优解的值 。
    c. 按自底向上的方式计算出最优解 。
    d. 由计算结果构建出一个值 。

装配线的调度

两条装配线 S₁ ，S₂ ，每个转配线上有 n 个装配点 。
S(i,j): 装配线1，装配点j.
a(i,j): 装配线1, 装配点j花费时间。
e(i): 进入装配线i花费时间
x(i): 离开装配线i花费时间。
t(i,j): 底盘从装配线 i 离开的时间 。

STEP 1 : 通过工厂最快路线的结构

通过 S₁ 装配线 j 装配点的最快路线 ，有两种可能 ：
a. 通过装配点 S(1,j-1) 的最快路线 ，直接通过 S(1,j)
b. 通过装配点 S(2, j-1) 的最快路线，由 S2 转到 S1 。

镜像的考虑 ，当通过 S2 的装配点 j 有
a. 通过装配点 S(2,j-1) 的最快路线 ，直接通过 S(2,j)
b. 通过装配点 S(1, j-1) 的最快路线，由 S1 转到 S2 。

STEP 2 : 构建一个递归解

f* = min(f₁[n] + x₁, f₂[n] + x₂)

f₁[j] = e₁ + a₁,₁ (j==1)
f₁[j] = min( f₁[j-1] + a(1,j) , f₂[j-1] + t(2,j-1) + a(1,j) )  (j>1)

f₂[j] = e₂ + a₂,₁ (j==1)
f₂[j] = min( f₂[j-1] + a(2,j), f₁[j-1]  + t(1,j-1) + a(2,j)) (j>1)

STEP 3 : 计算出一个最快时间

FAST-WAY(a, t, e, x, n)
  f₁[1] = e₁ + a(1,1)
  f₂[1] = e₁ + a(1,1)
  for j ← 2 to n 
    if f₁[j-1] + a(i,j) ≤ f₂[j-1] + t(2,j-1) + a(1,j)
        f₁[j] ← f₁[j] + a(i,j)
        l₁[j] ← 1
    else
        f₁[j] ← f₂[j-1] + t(2,j-1) + a(i,j)
        l₁[j] ← 2

    if f₂[j-1] + a(2,j) ≤ f₁[j-1] + t(1,j-1) + a(2,j)
        f₂[j] ← f₂[j-1] + a(2,j)
        l₂[j] ← 2
    else
        f₂[j] ← f₁[j-1] + t(1,j-1) + a(2,j)

  if f₁[n] + X₁ ≤ f₂[n] + x₂
    f* ← f₁[n] + x₁
    l* ← 1
  else
    f* ← f₂[n] + x₂
    l* ← 2
