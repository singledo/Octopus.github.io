算法导论十二章 二叉查找树

二叉查找树的基本性质

  • 查找树是一种数据结构，支持多种多台集合操作，包括 SEARCH, DELETE, MININUM, MAXINUM, PREDECESSOR, SUCCESSOR, INSERT 等，且时间为 Ѳ(lg n) — 完全二叉树。
  • 根节点的 key 值是大于左子节点的 key 值，小于右子节点的 key 值。
  • 数据结构一个由 key, satellite date, left children pointer, right children pointer 组成 .

  • 遍历二叉树的三种方法.

    前序遍历：根节点关键字在其左右子节点之前；
    中序遍历：根节点关键字在左右子节点之间；
    后序遍历：根节点关键之在左右子节点之后。

前趋和后继

在树中寻找后继节点时，先会判断当前节点的右节点是否存在. 如果右节点存在，去找一右节点为根节点的树中最小值。

则此值为当前节点的后继。如果当前节点没有右子节点，则应当回溯父亲节点，找到某个节点是其父亲节点的左节点结束。
找到的节点叫做最低先祖节点，同时也是要寻找的后继节点。

插入和删除

在执行插入操作时，会根据将要插入的节点的 key 值来确认他的父亲节点，在用插入点的 key 值和父亲节点的 key 值比较，大于父亲节点 key 值，

插入父亲节点右子节点，小于父亲节点子节点，插入父亲节点左子节点。
在执行删除操作时，会先判断需要删除节点的子节点个数。如果删除节点没有子节点，则将删除节点直接删除即可。如果删除节点右一个子节点，则
将删除节点的父亲节点指向删除节点的子节点，这样就将节点从树中删除了。如果删除节点右两个子节点，则会删除这节点的后继节点，然后用后继节
点的数据覆盖删除节点。
